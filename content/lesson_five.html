Difference between ERC 20 and ERC 721 ( part 1 )

ERC 20, ERC 721, ERC 1155

အပေါ်ကသုံးခုတော့ token standard လို့ခေါ်တဲ့ smartcontract တွေရေးသားရာမှာအသုံးပြုတဲ့standard တွေပါ ERC 20 token standard ကတော့ fungible token တွေအတွက် သူက ကိုယ်စားပြုပါတယ် fungible asset ဆိုတာမျိုး ကဘယ်လိုမျိုးလဲဆိုတော့ ဥပမာ မြန်မာကျပ်ငွေ တထောင် တန် တရွက် ကို တထောင်တန်နဲ့အမြဲတမ်းလဲလို့ ရတယ် လေ လဲလှယ်လို့ရတယ့်ပစ္စည်းပေါ့ဗျာ အလွယ်မှတ်ရင် အဲလိုမျိုးဘဲ usdc, dogecoin, sandbox တို့ mana တို့ဆိုတာ လဲ fungible assetလို့ခေါ်ပါတယ် crypto မှာ non-fungible asset တွေကတော့ NFT တွေပါဘဲ dogecoin ပိုင်တဲ့ owner နှစိယောက် က အချင်းချင်း လဲသုံးလို့ရပေမယ့် NFT တခုစီပိုင်ထားတဲ့ owner နှစ်ယောက်က လဲလို့အဆင်မပြေဘူးလေ NFT တွေကို တော့ ERC 721 standard ကိုလိုက်နာတဲ့ smartcontractတွေကိုအသုံးပြုပီးဖန်တီးကြတယ် nft တွေက ဘာလို့ non-fungibleဖြစ်တာလဲဆိုတော့ တခုနဲ့တခုက value မတူယုံတင်မကဘူး property လဲမတူဘူး ဥပမာ black label နဲ့ gold label ကစျေးတူပီး သုံးသောင်းခွဲ ဘဲပေးရတယ်ဆိုပါစို့ မောင် Aမှာ black label ရှိပီးတော့ မောင် B မှာ gold label ရှ်တယ် ဆိုပါစို့ မောင် A က မောင် B ရဲ့ gold နဲ့ လဲ သောက်ချင်တယ်ဆိုပါစို့ နှစ်ယောက်လုံး ကသဘောတူရင်တော့လဲလို့ရတယ် ဒါပေမယ့်အချိန်တိုင်း လဲလို့အဆင်မပြေဘူးလေ gold label နဲ့ black label က စျေးတူရင်တောင် property က မတူဘူးလေ non-fungible token တွေလဲဒီသဘောဘဲ
