Web 1.0, Web 2.0 and Web 3.0 ပထမ web 1ခေတ်တုန်းက user တွေက data တွေကို read လုပ်လို့ဘဲရခဲ့တယ် ဘယ်လိုပုံစံ မျိုးလဲ ဆိုတော့ web 1 တုန်း က website တွေမှာသူတို့ တင်တဲ့ informationတွေကိုဖတ်လို့ဘဲ ရသလိုမျိုးuser ကနေ ဓာတ်ပုံတို့ ဗီဒီယိုတို့ တခြား file တွေကို write လုပ်လို့မရဘူး ဒုတိယ web 2 အခုကျွန်တော် တို့သုံးနေတာပေါ့ user က write လုပ်လို့ရလာတယ် ဥပမာဆို social mediaကိုဘဲကြည့် videoတင်လို့ရတယ် ပုံတွေတင်လို့ရလာတယ် တတိယ နောက် ပေါ်လာမယ့် web 3 မှာတဆင့် ထပ်မြှင့်သွားပီ web 2မှာက ကျွန်တော် တို့ လို user တွေက writeလုပ်လို့ရပေမယ့် အဲ data တွေကိုတကယ်မပိုင်ပါဘူး facebook တို့ google, tencentတို့လို လက်တဆုပ်စာ company ကြီးတွေကဘဲပိုင်တာပါ but web 3 ရောက်ရင် ပိုင်လာမှာပါ ဥပမာ ဆို pubg မှာကျွန်တော်တို့ရထားတဲ့ပိုင်တဲ့ game item ကကျွန်တော်တို့ ပိုင် တယ် လို့ ထင်ရပေမယ့် တကယ်က companyကပိုင်တာပါ တခြား pubgလိုဂိမ်းနောက်တခုမှာ အဲ game item ကိုသုံးလို့မရဘူး but web 3ရောက်ရင်တော့ blockchain နည်းပညာအကူအညီနဲ့ ကိုယ့် game item ကိုgame တခုကနေ နောက်တခုကို ပြောင်းသုံးလို့ရတာမျိုးထိဖြစ်လာနိုင်ပါတယ် ကိုယ့် game item ကိုအဲဒါမှ ကိုယ်တကယ်ပိုင်သလိုဖြစ်တာပါ web3မှာ အဲလိုဖြစ်လာနိုင်ပါတယ်
